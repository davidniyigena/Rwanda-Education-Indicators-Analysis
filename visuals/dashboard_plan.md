# Dashboard Plan: Rwanda Education Indicators Analysis

## Dashboard Purpose

This dashboard is designed to summarize Rwanda education indicators and help users understand trends in education access, completion, equity, and quality over time.

The dashboard will support education monitoring, policy analysis, program planning, and evidence-based decision-making.

---

## Dashboard Users

The intended users of this dashboard include:

- Education data analysts
- Monitoring and evaluation specialists
- Education program managers
- Policy researchers
- Development partners
- Nonprofit organizations
- Government education stakeholders

---

## Key Performance Indicators

The dashboard may include the following KPI cards:

- Latest primary enrollment rate
- Latest secondary enrollment rate
- Latest primary completion rate
- Latest secondary completion rate
- Latest primary gender parity index
- Latest secondary gender parity index
- Latest primary pupil-teacher ratio
- Number of indicators analyzed

---

## Planned Dashboard Sections

### 1. Executive Summary

This section will provide a high-level overview of Rwanda education indicator trends, including enrollment, completion, gender parity, and pupil-teacher ratio.

### 2. Enrollment Trends

This section will show changes in primary and secondary school enrollment rates over time.

### 3. Completion Trends

This section will compare primary and secondary completion rates across years.

### 4. Gender Parity Trends

This section will show gender parity trends for primary and secondary education.

### 5. Education Quality Indicator

This section will track the primary pupil-teacher ratio as an education quality-related indicator.

### 6. Category Summary

This section will summarize average indicator values by category, including access, completion, equity, and quality.

---

## Planned Visualizations

The dashboard may include:

- Line chart of enrollment trends
- Line chart of completion rate trends
- Line chart of gender parity trends
- Line chart of pupil-teacher ratio trend
- Bar chart of average indicator values by category
- KPI cards for latest indicator values
- Summary table of indicators

---

## Suggested Dashboard Filters

The dashboard may include filters for:

- Year
- Indicator name
- Education level
- Category
- Unit

---

## Expected Dashboard Insights

The dashboard is expected to help answer questions such as:

1. How have primary and secondary enrollment rates changed over time?
2. How have completion rates changed over time?
3. Has gender parity improved in education?
4. Is the pupil-teacher ratio improving?
5. Which education categories show stronger or weaker performance?
6. What trends may require continued education policy or program support?
